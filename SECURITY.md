## Security Policy

### Reporting a Vulnerability

If you believe you have found a security vulnerability in DirectiveOps Scanner:

- Please do not open a public issue.
- Instead, report it privately via the security contact listed in the GitHub repository security settings or by opening a GitHub Security Advisory.
- Include as much detail as possible to help us reproduce and assess the issue.

We will acknowledge receipt, investigate promptly, and communicate next steps as we triage and address the report.
